# Password-Strengh-Checker
This is a program to check user's password strength and give suggestions on how to improve it

# Features:
- Checks password length (minimum 8 characters)
- Detects uppercase letters
- Detects lowercase letters
- Detects numbers
- Detects special characters
- Displays password strength as Weak / Medium / Strong
- Gives improvement remarks if password is not strong enough

## How to Run
1. Install Python on your system (any version above 3.6)
2. Run the script
3. Input the password when prompted.
